# Weline Acl 模块 - 事件文档

## 概述

本文档详细说明了 Weline Acl 模块提供的 `Weline_Acl::no_access_redirect_before` 事件及其使用方法。该事件在用户无权限访问时，执行重定向操作前触发。

## 事件列表

### 1. Weline_Acl::no_access_redirect_before - 无权限访问重定向前事件

#### 基本信息

- **事件名称**：`Weline_Acl::no_access_redirect_before`
- **事件类型**：ACL权限事件
- **触发时机**：在用户无权限访问时，执行重定向操作前
- **触发位置**：`app/code/Weline/Acl/Observer/RouteBefore.php` 第 88 行
- **配置文件**：`app/code/Weline/Acl/etc/event.xml`

#### 功能说明

`Weline_Acl::no_access_redirect_before` 事件在用户无权限访问时，执行重定向操作前触发。允许其他模块在重定向前执行自定义操作，如记录日志、发送通知等。

该事件主要用于：
- 记录无权限访问日志
- 发送安全通知
- 自定义重定向逻辑
- 权限审计

#### 触发时机

```php
// app/code/Weline/Acl/Observer/RouteBefore.php
$event = ObjectManager::getInstance(EventsManager::class);
$event->dispatch('Weline_Acl::no_access_redirect_before');
$request->_response->noRouter(DEV ? 403 : 404);
```

#### 使用场景

- **日志记录**：记录无权限访问的详细信息
- **安全通知**：向管理员发送安全警报
- **审计追踪**：记录权限检查失败的情况
- **自定义处理**：执行自定义的无权限访问处理逻辑

#### 使用方法

##### 基本用法

在模块的 `etc/event.xml` 文件中注册观察者：

```xml
<?xml version="1.0"?>
<config xmlns:xs="http://www.w3.org/2001/XMLSchema-instance"
        xs:noNamespaceSchemaLocation="urn:Weline_Framework::Event/etc/xsd/event.xsd"
        xmlns="urn:Weline_Framework::Event/etc/xsd/event.xsd">
    <event name="Weline_Acl::no_access_redirect_before">
        <observer name="Your_Module::no_access_redirect_before"
                  instance="Your\Module\Observer\NoAccessRedirectBeforeObserver"
                  disabled="false"
                  shared="true"
                  sort="100"/>
    </event>
</config>
```

创建观察者类：

```php
<?php

namespace Your\Module\Observer;

use Weline\Framework\Event\Event;
use Weline\Framework\Event\ObserverInterface;

class NoAccessRedirectBeforeObserver implements ObserverInterface
{
    /**
     * 执行观察者逻辑
     *
     * @param Event $event
     * @return void
     */
    public function execute(Event &$event): void
    {
        // 记录无权限访问日志
        // 发送安全通知
        // 执行自定义处理逻辑
    }
}
```

#### 事件数据

`Weline_Acl::no_access_redirect_before` 事件不传递额外的数据。

#### 注意事项

1. **执行顺序**：观察者按照 `sort` 属性排序执行，数值越小越先执行
2. **性能考虑**：该事件在每次无权限访问时都会触发，应避免执行耗时操作
3. **不要阻止重定向**：该事件在重定向前触发，观察者不应阻止后续的重定向操作
