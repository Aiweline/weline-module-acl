# Weline Acl 模块 - 事件文档

## 概述

本文档详细说明了 Weline Acl 模块提供的 `Weline_Acl::check_role` 事件及其使用方法。该事件在检查用户角色权限时触发。

## 事件列表

### 1. Weline_Acl::check_role - 角色检查事件

#### 基本信息

- **事件名称**：`Weline_Acl::check_role`
- **事件类型**：ACL权限事件
- **触发时机**：在检查用户角色权限时
- **配置文件**：`app/code/Weline/Acl/etc/event.xml`

#### 功能说明

`Weline_Acl::check_role` 事件在检查用户角色权限时触发，允许其他模块自定义角色检查逻辑。

该事件主要用于：
- 自定义角色检查逻辑
- 角色权限扩展
- 角色权限验证
- 角色权限日志记录

#### 使用场景

- **角色检查**：执行自定义角色检查逻辑
- **权限验证**：验证用户角色权限
- **权限扩展**：扩展角色权限功能
- **日志记录**：记录角色检查日志

#### 使用方法

##### 基本用法

在模块的 `etc/event.xml` 文件中注册观察者：

```xml
<?xml version="1.0"?>
<config xmlns:xs="http://www.w3.org/2001/XMLSchema-instance"
        xs:noNamespaceSchemaLocation="urn:Weline_Framework::Event/etc/xsd/event.xsd"
        xmlns="urn:Weline_Framework::Event/etc/xsd/event.xsd">
    <event name="Weline_Acl::check_role">
        <observer name="Your_Module::check_role"
                  instance="Your\Module\Observer\CheckRoleObserver"
                  disabled="false"
                  shared="true"
                  sort="100"/>
    </event>
</config>
```

#### 事件数据

`Weline_Acl::check_role` 事件传递的数据根据具体实现而定。
